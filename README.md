# BetterTimeStop

## 功能描述

### 在基地外
- 打开背包、地图、商人、分解和 ATM 等菜单时，触发全局时停（包括敌人和子弹）
- 其余时间正常流动
- 舔包时物品加载过程中时间不会暂停，需等待物品全部加载完成后才会时停
- **建议**：清空物品后再进行舔包操作

### 在基地内
- 默认时间正常流逝
- 打开地图可切换基地时间状态：
  - 首次打开地图：基地时间暂停
  - 再次打开地图：基地时间恢复正常
- 时停状态下仍可与 NPC 进行交互

## 兼容性说明

### 可能存在冲突
- 与修改游戏时间流速相关的 Mod
- 与修改物品搜索时长、音效、种类相关的 Mod（可能产生 bug，也可能正常运行）

### 应该无冲突
- 显示分解信息类的 Mod
- 任务道具相关的 Mod
